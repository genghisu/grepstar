How can this site help me?

Django360 is a full featured knowledge sharing site dedicated to the helping people find answers to their Django related questions. If you got a question, check out the tutorials, code snippets and apps to see if someone has already thought of a solution. If not, ask a question and get it answered by our knowledgeable community.

What features does this site have?

On Django360, you can ask and answer questions. We also let users submit Django apps, code snippets and Django related tutorials. Everything is peer moderated through voting and flagging of content.

What kind of content should be post here?

You should submit Django related content. Questions concerning all aspects of Django development are welcome. Please submit high quality apps and code snippets.

What kind of content should not be posted here?

Advertisement. Subjective questions. Bug ridden code snippets.

Can I use the site anonymously?

Sure, everything on the site can be browsed as an anonymous user. If you want to access the community features such as voting and commenting, then you will have to login via OpenID or create an account.

What is reputation?

Reputation is an indication of how well trusted an user is on Django360. Users with high reputations can access higher level moderation features of the site such as editing any piece of content, access to comments, moderation features, etc. Reputation can be gained by having useful content upvoted by other users.
